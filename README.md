# StudyHiveHub

A gamified, AI-powered learning platform with micro-lessons, community collaboration, and interactive visual notes.

## Features

- Gamified Learning Rewards (points, badges, streaks)
- AI-Powered Personalized Study Plans
- Micro-Learning “Snacks”
- Community-Driven Q&A & Collaboration
- Interactive Visual Notes
- Limited-Time Study Challenges
- User-Generated Mini-Courses
